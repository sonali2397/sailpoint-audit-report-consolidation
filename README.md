# :rocket: SailPoint Audit Report Consolidation Framework

## :pushpin: Overview
A Python-based automation framework designed to **consolidate and standardize SailPoint IdentityIQ certification campaign reports** for audit and compliance workflows.

The project automates extraction, merging, and processing of certification campaign data from multiple reports to generate a **unified audit-ready dataset**, improving efficiency in Identity Governance (IGA) operations.

---

## :dart: Key Features
- Automated consolidation of SailPoint certification campaign reports
- Merging of multiple campaign status datasets into a unified output
- Structured processing of CSV-based reporting artifacts
- Generation of audit-ready compliance reports
- Reduced manual intervention in report aggregation workflows
- Improved consistency in Identity Governance reporting
